=========
Model API
=========

This API layer provides access to the higher-level objects stored in Github
repositories.  A single Github repository serves as the persistent storage for
the CMS.  All data is fetched from Github through the :doc:`Remote API
<remote_api>` layer and turned into formal objects by this model layer.
